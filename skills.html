Habilidades

Estas son algunas de las tecnologías y herramientas con las que he trabajado y que sigo aprendiendo como desarrollador de software.

Lenguajes de Programación

Java

Java es un lenguaje de programación de alto nivel utilizado para el desarrollo de aplicaciones de escritorio, web y móviles. Me siento cómodo trabajando con Java y desarrollando aplicaciones complejas con este lenguaje.

Python

Python es un lenguaje versátil y fácil de aprender. Actualmente, estoy en proceso de aprender Python para ampliar mis habilidades en áreas como el análisis de datos y la inteligencia artificial.

JavaScript

JavaScript es un lenguaje fundamental para el desarrollo web. Estoy aprendiendo JavaScript para crear interfaces interactivas y mejorar la experiencia de usuario en aplicaciones web.

Bases de Datos

MySQL

MySQL es un sistema de gestión de bases de datos relacional. Tengo experiencia trabajando con MySQL para administrar y consultar datos de manera eficiente en aplicaciones de software.

Herramientas de Control de Versiones

Git, GitHub y GitLab

Git es una herramienta de control de versiones que permite gestionar y guardar cambios en proyectos de software. Utilizo Git en conjunto con plataformas como GitHub y GitLab para colaborar en proyectos y mantener un historial de versiones.

IDEs y Editores

Eclipse

Eclipse es un entorno de desarrollo integrado (IDE) popular para el desarrollo en Java. Lo utilizo principalmente para mis proyectos de Java, ya que ofrece muchas herramientas útiles para el desarrollo de software.

Visual Studio Code

Visual Studio Code es un editor de código versátil y ligero que utilizo para desarrollo web y otros lenguajes de programación. Su compatibilidad con extensiones hace que sea muy útil para trabajar con diferentes tecnologías.

NetBeans

NetBeans es otro IDE que utilizo, especialmente para proyectos en Java y desarrollo web. Ofrece una buena integración con herramientas de desarrollo y es fácil de usar.

Otros

Diseño de Interfaces Gráficas y UX/UI

Tengo conocimientos en diseño de interfaces gráficas y principios de UX/UI, lo que me permite crear aplicaciones con interfaces intuitivas y orientadas a mejorar la experiencia del usuario.
